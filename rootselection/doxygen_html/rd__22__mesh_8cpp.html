◆ rootTimer()

bool rootTimer ( )

Timer that resets when keep alive message from root has been received.

We can use this to check if there is a root node in the network. The root node will send a broadcast at set intervals. We have to selest a new node when there is no more root.

Returns
true No message of the root has been received in the timeframe that has been set.
false A message has been received in the timeframe.
