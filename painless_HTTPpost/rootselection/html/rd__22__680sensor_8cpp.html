String makeSensorMessage680 ( bool  isRoot,
String  nodeName,
String  IP,
String  nodeID 
)

Make the JSON that we send to the root node.

This JSON is based on the SenML specification. This one JSON will include multiple measurements.

bn: Base name or sensor name (sensor1)

n: Measurement name (temperature)

u: Unit of Measurement (°C)

v: Value (21.8)

Parameters
isRootIf we are root we also send the IP address of the root so Home Assistant can send API requests to the root.
nodeNameThis is part of the entity ID we want to use in Home Assistant
IPIP address of the node.
nodeIDNodeID, we use this to check duplicate names in Home Assistant
Returns
String String in JSON format with all the measurements
